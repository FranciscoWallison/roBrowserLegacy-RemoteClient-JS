Setup Instructions

This is a remote client server for roBrowser. To get started:

  1. Place your GRF files in the resources/ directory
  2. Edit the resources/DATA.INI file to list your GRF files: 
    [Data]
0=rdata.grf
1=data.grf
  3. Copy your BGM/, data/, and System/ folders to the project root
  4. Configure src/config/configs.js as needed
  5. Restart the server
Note: Make sure your GRF files are version 0x200 without DES encryption. Use GRF Builder to repack if needed: Options > Repack type > Decrypt -> Repack

API Endpoints

  • GET /list-files - List all available files
  • POST /search - Search for files (if enabled)
  • GET /* - Get any file from GRF or filesystem
